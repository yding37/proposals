# Yi publication PDFs

This folder stores publication metadata and downloaded PDFs for the CISE - Small Reasoning Models proposal.

Files:

- `citations.csv`: Google Scholar publication export supplied by Yi Ding.
- `download_yi_publication_pdfs.py`: downloader that searches public PDF sources.
- `download_manifest.csv`: generated by the downloader, showing the status and source URL for each publication.
- `*.pdf`: downloaded papers and patent PDFs when public PDFs are found.

The repository workflow `.github/workflows/download-yi-papers.yml` can be run manually from GitHub Actions. It downloads available PDFs and commits them back into this directory.

For better DOI/open-access lookup, add a repository secret named `UNPAYWALL_EMAIL` with an academic email address.
